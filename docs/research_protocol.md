# Research Protocol

1. Problem discovery — define verifiable task families.
2. Literature sweep — find prior art (Kaggle notebook 01).
3. HF duplication sweep — check models, datasets, spaces (notebook 01).
4. OpenRouter model sweep — identify eligible models (notebook 02).
5. Legitimacy QA — score across 10 dimensions; accept >= 85.
6. Decision record — accept, revise, or reject.

All live searches run on Kaggle. No local network calls.
