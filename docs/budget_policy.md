# Budget Policy

- Max total spend: US$4.00
- Target: below US$2.00
- Max input price: US$0.12/1M tokens (exclusive)
- Max output price: US$0.30/1M tokens (exclusive)
- dry_run default: true
- stop_on_unknown_price: true
- stop_on_budget_exceeded: true
- require_price_verification: true

No call without a prior cost estimate. No model with unverified price.
