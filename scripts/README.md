# Scripts

These scripts are designed to run on Kaggle. Do not run them locally.

## Scripts
- `run_kaggle_eval.py` — run an evaluation battery (dry or real).
- `build_leaderboard.py` — build a leaderboard from an evaluation report.
- `package_hf_dataset.py` — package the HF dataset repo.
- `package_hf_space.py` — package the HF Space repo.
